Como extrair o arquivo N27 para o pendrive

Para extrair o arquivo N27 para o pendrive, conecte o pendrive ao teclado, acesse o Menu > Menu Dois > Expansion > Export Instrument Info, confirme e o arquivo será salvo no pendrive.

Passo a Passo Detalhado:

1. Conecte o Pendrive: Insira o pendrive em uma porta USB do seu teclado Yamaha.

2. Acesse o Menu: Pressione o botão Menu ou Function no teclado.

3. Navegue para o Menu Dois: Use os botões direcionais (ou o botão Tab) para ir até o Menu Dois (Menu 2).

4. Encontre "Expansion": Gire o botão de função ou use as setas para encontrar a opção Expansion e pressione Enter.

5. Selecione "Export Instrument Info": Dentro de "Expansion", escolha a opção Export Instrument Info (Exportar Informações do Instrumento) e pressione Enter.

6. Confirme a Exportação: Aperte OK ou o botão correspondente (como o botão 7 para baixo ou letra correspondente) para confirmar.

7. Verifique no Pendrive: O arquivo N27 (com o nome do seu teclado, ex: PSR-SX900_IntrumentInfo.n27) será salvo no pendrive.

Próximos Passos (Após exportar):

1. Envie o arquivo: Conecte o pendrive ao computador, copie o arquivo N27 e envie-o por e-mail ou WhatsApp para que possamos criar o pack de sons para você.

2. Instalação: Depois, quando receber o pack, você conecta o pendrive de volta e realiza o processo inverso (importação/instalação).
